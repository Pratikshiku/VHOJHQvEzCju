### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 商品信息
5. 商品资讯
6. 个人中心
7. 购物车
8. 在线客服
9. 我的订单
10. 我的地址
11. 我的收藏
12. 修改密码
13. 订单评价管理
14. 我的收藏管理
15. 我的订单管理


**管理员功能：**

2. 个人中心
